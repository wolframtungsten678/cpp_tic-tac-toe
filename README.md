# **Tic-Tac-Toe**
## **What Is This?**
This is a simple terminal-based tic-tac-toe game. Follow the terminal prompts to select a square for each player's turn. 

## **System Requirements**
g++ must be installed

## **How To Run The Game**
The tic-tac-toe game can be run from the command line: 
````bash
./run.sh
````
